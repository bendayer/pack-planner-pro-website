## Phase 1 — Measurement: FAILING

**No key events are marked in GA4.**

The GA4 key events section is completely empty. `begin_checkout`, `ppp_purchase_click`, and `generate_lead` are all firing as regular events but none have been marked as key events in GA4 admin.

This means:
- Cannot tie SEO performance to commercial outcomes
- Cannot see which landing pages drive buying intent
- Conversion rate is untrackable

**Action required:** Log into GA4 → Admin → Key Events → mark these three:
- `begin_checkout`
- `ppp_purchase_click`
- `generate_lead`

This is Phase 1 and everything else is harder to judge without it.

---

## Phase 2 — Technical: CONFIRMED ISSUE

**Contact route split is live in Search Console.**

Both URLs are indexed and appearing for navigational queries:
- `/contact.html` — position 2, 1 click, 4 impressions
- `/contact` — position 2, 0 clicks, 4 impressions

Google is splitting signals between two routes for the same page. The canonical on `/contact.html` should be confirmed as the single public URL, and the `/contact` rewrite should redirect rather than serve a duplicate.

**Action required:** Add a redirect in vercel.json from `/contact` to `/contact.html`, or ensure the rewrite serves the correct canonical. (Note: do NOT add www-to-non-www redirects — previous outage risk.)

## Phase 3 — CTR: ONE CRITICAL OPPORTUNITY

### The biggest finding in this data

**`/dog-walking-software-no-monthly-fee` is at position 5.25 with 0 clicks from 8 impressions.**

Position 5 should get clicks. This page is ranking in the top 5 for some queries and generating zero interest in the snippet. This is a title and description problem, not a ranking problem.

Current title: `Dog Walking Software No Monthly Fee | Pack Planner Pro`
Current description: unknown — needs checking against live page

Per the SEO system, the formula is:
- Title: `[Primary Keyword] | Pack Planner Pro — [Differentiator]` under 60 chars
- Description: Pain → Solution → Differentiator → CTA, price included, under 155 chars

**Suggested new title:**
`Dog Walking Software — No Monthly Fee | Pack Planner Pro` (56 chars)

**Suggested new description:**
`Tired of monthly software bills? Pack Planner Pro is a one-time £79 Windows app for UK dog walkers. No subscription. No account. Yours to keep.` (144 chars)

This is the single highest-impact change available right now.
