## App vs Software

Research shows two distinct search behaviours:

- **"App"** — broader, mobile-first, higher search volume
- **"Software"** — more business-focused, higher purchase intent

### Action

Add "app" language alongside "software" on the homepage and `/dog-walking-software-uk`.

Do not create a separate `/dog-walking-app-uk` page until Search Console confirms meaningful demand.

## Comparison-Driven Segment

Buyers in this market search by comparison:

- best dog walking software UK
- dog walking software reviews
- dog walking software vs spreadsheets
- alternatives to [competitor name]

### What this segment wants

They are not just looking for features. They want to know why Pack Planner Pro is a better choice than whatever they are currently using or have found first.

### Action

- Add comparison framing and objection-handling copy to existing pages
- Add FAQ entries that address comparison queries directly
- Consider a dedicated `/best-dog-walking-software-uk` page once the core landing pages are strong and Search Console shows repeat demand on comparison queries

## New Page Decision Rules

Create a new page only if all three of these are true:

1. Search Console shows repeated demand for a theme not covered well by existing pages.
2. The new page can carry a genuinely distinct intent — not a slight keyword variation.
3. There is enough real product detail to make it useful, not thin.
