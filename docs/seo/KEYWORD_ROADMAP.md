## Content Rules

- Refresh before creating anything new.
- Each page needs one clear intent.
- Do not publish thin keyword variations that repeat the same story.
- Use feature proof, screenshots, FAQs, and objection handling to support intent.

## New Page Criteria

Create a new page only if all of these are true:

1. Search Console shows repeated demand for a theme that does not fit the current pages well.
2. The new page can carry a distinct intent, not just a slight keyword variation.
3. There is enough real product detail to make the page genuinely useful.
